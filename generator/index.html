IPTC News in JSON Generator: ninjs 1.3 and 2.0

This tool shows the capabilities of IPTC's ninjs standard for news in JSON. As you enter story metadata into the form, you can see the corresponding ninjs JSON appear. You can switch between ninjs 1.3 and 2.0 output using the selector above the JSON output.

For more information about ninjs, see the IPTC ninjs pages or the ninjs User Guide.
